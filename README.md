# Game of Liars

A real-time multiplayer bluffing game where teams compete through strategic deception and answer stealing.

## 🎮 Game Overview

**Game of Liars** is a web-based multiplayer game where two teams (Blue vs Red) compete by answering questions and strategically stealing points from each other. The twist? Teams can bluff about their answers, and captains can steal points if they think the other team got it right!

## 🎯 How to Play

### Game Flow
1. **Create/Join Lobby** - Enter your name and join with a 4-digit code
2. **Team Selection** - Choose Blue or Red team as Captain or Member
3. **Pre-game Setup** - Host configures game settings (rounds, time limits)
4. **Game Rounds** - Teams answer questions and can steal points

### Scoring System
- **Correct Answer**: 1 point for your team
- **Wrong Answer**: 0 points
- **Stealing**: If other team got it right, stealer gets 1 point
- **Bluffing**: Teams can lie about their answers to mislead opponents

### Roles
- **Host**: Controls game settings and adjudicates ties
- **Captain**: Submits final answers for their team
- **Member**: Contributes to team discussion and strategy

## 🚀 Tech Stack

- **Frontend**: React.js with Tailwind CSS
- **Backend**: Node.js with Express
- **Real-time**: Socket.io for live updates
- **Database**: MongoDB for persistent data
- **Authentication**: Session-based user management


## 🏗️ Key Features

- **Real-time Team Chat**: Coordinate with teammates
- **Live Score Updates**: Instant point tracking
- **Timer System**: Round-based time limits
- **Host Controls**: Game settings and adjudication
- **Responsive Design**: Works on desktop and mobile
